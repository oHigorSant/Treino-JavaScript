# Treino JavaScript

Exercícios de JavaScript puro focados em manipulação de dados: listas, objetos, filtragem e transformação de arrays.

## Arquivos

| Arquivo | Conteúdo |
|---------|----------|
| `ManipularDados.js` | Exercícios de manipulação de arrays e objetos |
| `RespostaLista1.js` | Respostas da lista de exercícios 1 |

## Conceitos praticados

- Métodos de array: `map`, `filter`, `reduce`, `find`
- Manipulação de objetos
- Funções e escopo

## Tecnologias

- JavaScript (Node.js ou browser console)
